# สวนหย่อนใจ — เวอร์ชันปรับปรุง (พี่บุก / สายลุย)

เธอบอกแค่ว่า "ห้องนี้ชอบน้อยที่สุด" แต่ไม่ได้บอกว่าตรงไหน ฉันเลยเปิดของจริงดูเอง แล้ววินิจฉัยตามนี้ — ทำเป็นเดโมจับได้จริงมาให้แล้ว เปิดดับเบิลคลิกได้เลย

---

## 1. วินิจฉัย — ห้องนี้อ่อนตรงไหน (ตรงประเด็น)

**ปัญหาใหญ่สุด: ห้องนี้ให้ความรู้สึกเป็น "เกมในห้าง" ไม่ใช่ "สวนฮีลใจ"**

1. **เกมช้อปสบายใจคือตัวฉุดที่สุด** — มันคือเกมแตะ emoji ใส่ตะกร้าแข่งกับเวลา (15/30/60 วิ) มี timer นับถอยหลัง มี HUD เหมือนเกมมือถือโฆษณา อันนี้ขัดกับ AI_BRIEF ข้อ 1 ตรง ๆ ที่เขียนว่า "ไม่ใช่ challenge / TikTok energy" — การแตะรัว ๆ แข่งเวลา = TikTok energy เป๊ะ ฉันเชื่อว่านี่แหละคือสาเหตุหลักที่เธอรู้สึกไม่เข้ากับห้องอื่น

2. **คำเปิดห้องกันคนออก** — subtitle เดิมเขียนว่า *"ใจชอบ Extrovert ห้องนี้สำหรับเธอ"* แปลว่าถ้าเป็น introvert / วันที่ไม่มีแรง = ห้องนี้ไม่ใช่ของเธอ ขัดกับโทนโอบกอดทุกคนของแบรนด์

3. **มีฟีเจอร์ 2 อันไม่เกี่ยวกันวางซ้อน** — วงล้อ + เกมช้อป คนละเรื่อง คนละอารมณ์ อยู่หน้าเดียวกัน ทำให้ห้องไม่มีแก่น

4. **กดเลือกเยอะก่อนได้เล่น** — เปิดมาต้องเลือก "List ทีมเลือก / เขียนเอง" ก่อน ทั้งที่ฮีลใจควรกดน้อย ๆ อยากหมุนก็หมุนเลย

5. **ผลลัพธ์จบห้วน** — หมุนเสร็จได้กิจกรรมแล้วก็... จบ ไม่มีการชวนต่อแบบอบอุ่น ทั้งที่ AI_BRIEF §8 ขอ "ถ่ายรูปแชร์หลังทำกิจกรรม" ไว้แล้วแต่ยังไม่มี

---

## 2. ฉันตัดสินใจทำอะไร (ลงมือเลย ไม่โยนให้เธอเลือก)

| เปลี่ยน | จากเดิม | เพราะ |
|---|---|---|
| **ตัดเกมช้อปแตะ-แข่งเวลาออก** | เกม arcade แตะ emoji | นอกแบรนด์ที่สุด ฉันยกออกจากเดโม |
| **เพิ่ม "วันนี้ใจมีแรงแค่ไหน?"** (เบา ๆ / พอไหว / มีแรง) | ไม่มี | แทนที่จะกันคน introvert ออก → ให้ทุกคนเข้าได้ วันไหนหมดแรงก็ได้กิจกรรมเบา ๆ (มองท้องฟ้า / ชงเครื่องดื่มอุ่น) วันมีแรงค่อยได้ชวนเพื่อนออกทริป กดทีเดียว ไม่เพิ่มภาระ |
| **List กิจกรรมจริงแบ่ง 3 ระดับ** | list เดียวรวม ๆ | กิจกรรมตรงกับแรงใจวันนั้น = ทำได้จริงไม่กดดัน |
| **ตัดโหมดเลือก default/custom หน้าแรกออก** | ต้องเลือกก่อนเล่น | เปิดมาหมุนได้เลย ลดแรงเสียดทาน (custom ค่อยไปไว้เมนูรองในเว็บจริง) |
| **ผลลัพธ์ต่อยอดอบอุ่น 3 ปุ่ม** | จบห้วน | หมุนใหม่ / เก็บใส่ใจ (bookmark) / **ถ่ายรูปแชร์** ตามที่ §8 ขอ |
| **วงล้อสวยขึ้น + กลีบดอกร่วงตอนได้ผล** | วงล้อแบน ๆ | หมุนแล้วฟินขึ้น อบอุ่นขึ้น แต่ไม่หวือหวาเกินจนหลุดธีม |
| **คำพูดโอบกอด** | "ลองออกไปทำดูสักที่" | "ไม่ต้องทำให้ดี แค่ลองออกไปให้ใจได้ขยับก็พอ" — โทนเขื่อน/ดนัย |

ใช้สีแบรนด์จริง (terracotta/coral + cream/clay/sage) ดึงมาจาก styles.css ตรง ๆ, ใช้ system font ล้วน (ไม่มี Google Font / CDN ตามกฎ AI_BRIEF), responsive ปุ่มแตะใหญ่ทั้งหมด

## 3. อยากให้เธอลองกด/ดูในเดโม

1. **กดสลับ "เบา ๆ / พอไหว / มีแรง"** — วงล้อจะเปลี่ยนกิจกรรมตามแรงใจทันที (นี่คือหัวใจของการปรับ)
2. **กดหมุน** — ดูว่าหมุนลื่น มีกลีบดอกร่วงนุ่ม ๆ ตอนได้ผล
3. **ดูปุ่มผลลัพธ์ 3 ปุ่ม** — โดยเฉพาะ "ถ่ายรูปแชร์" (บนมือถือจริงจะเด้งเมนูแชร์ของเครื่อง)
4. **กด EN/TH มุมขวาบน** — สลับสองภาษาได้ทั้งห้อง วงล้อก็เปลี่ยนภาษาตาม
5. **เปิดบนมือถือ** — ปุ่มแตะง่าย ไม่ล้นจอ

## 4. ถ้าเธอชอบทิศนี้ ขั้นต่อไปจะเอาไปใส่ travel.html จริงยังไง

ฉันจะยกตัวเลือกแรงใจ 3 ระดับ + list ใหม่ + ปุ่มถ่ายรูปแชร์เข้าไปแทนของเดิมใน travel.html ใช้ common.js ที่มีอยู่ (share/bookmark/lang) ไม่สร้างใหม่ แล้วลบ section เกมช้อป (#shopSection) กับ JS ใน travel.js ออก — เธอแค่เคาะว่า "เอาทิศนี้" ฉันลงมือต่อให้เสร็จเอง
