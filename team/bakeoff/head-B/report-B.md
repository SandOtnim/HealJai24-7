# สวนหย่อนใจ — รายงานสายปรึกษา (Head B "พี่ปรึกษา")

## 1) วินิจฉัย — ห้องนี้อ่อนตรงไหน (เรียงหนัก→เบา)

ของจริงตอนนี้มี 2 ฟีเจอร์ในห้องเดียว: วงล้อสุ่มกิจกรรม + เกม "ช้อปสบายใจ" (แตะอิโมจิแข่งกับเวลา)

1. **เกมช้อปขัดโทนแบรนด์เต็มๆ** — AI_BRIEF เขียนชัด "ไม่ใช่ challenge/TikTok energy" แต่เกมช้อปมีนาฬิกานับถอยหลัง + ของไหลรัว + กดให้ทันเวลา = อารมณ์เกมกดที่สร้าง *ความกดดัน* ตรงข้ามกับฮีลใจ ฉันเดาว่านี่คือตัวที่ทำให้เธอไม่อิน
2. **สองฟีเจอร์ไม่เป็นเรื่องเดียวกัน** — หัวห้องบอก "ออกไปทำกิจกรรม Extrovert" แต่เกมช้อปคือนั่งจิ้มจอ คนละทิศ ห้องเลยเหมือนแปะ 2 อย่างมั่วๆ
3. **วงล้อจบห้วน** — ได้ข้อความแห้งบรรทัดเดียว ไม่มีน้ำเสียงโอบกอด ไม่บอกว่าดีกับใจยังไง และไม่มี loop ปิดท้าย (งานค้าง §8 อยากได้ "เล่าหลังทำ" แต่ปุ่มแชร์อยู่ก่อนลงมือ)
4. **ไม่มีความรู้สึกสะสม** — หมุนแล้วจบ ไม่มีเหตุผลกลับมาเล่นอีก
5. **มือถือ** — ตัวหนังสือในวงล้อ auto-ย่อตามความยาว พอ default list ยาว 12 ช่อง ฟอนต์เล็กจนอ่านยาก

**สรุป:** ของที่ "ใช่" คือวงล้อ (ตรงธีม) ตัวที่ทำห้องเพี้ยนคือเกมช้อปแบบจับเวลา

## 2) แผน — 3 ทาง

- **ทาง A ขัดเงาของเดิม เก็บทั้ง 2** — เสียของเดิมน้อย แต่แก่นเกมช้อปยังตื้นและไม่เข้าธีมอยู่ดี
- **ทาง B ยกวงล้อเป็นพระเอก พักเกมช้อปไว้ก่อน ⭐ แนะนำ** — ห้องมีแกนชัดเรื่องเดียว เข้าแบรนด์ 100% ตอบงานค้าง §8 ครบ · ข้อเสีย: ต้องเอาฟีเจอร์ทีมออก 1 อัน **(ปกติตรงนี้ฉันจะหยุดถามเธอก่อนว่าโอเคไหมที่จะพักเกมช้อป — ไม่กล้าเคาะเอง)**
- **ทาง C รื้อทั้งห้อง metaphor ใหม่** — สดใหม่แต่งานเยอะ เสี่ยงไม่ทัน 14 มิ.ย.

**แนะนำทาง B** เพราะแก้ปัญหาราก (อารมณ์ผิดแบรนด์ + ห้องไม่มีแกน) โดยไม่รื้อใหญ่

> หมายเหตุรอบประชัน: ปกติฉันจะหยุดให้เธอเคาะก่อน แต่รอบนี้เดินตามทาง B ที่ตัวเองแนะนำไปเลย ทำเป็นเดโมจริง

## 3) อยากให้เธอลองกด/ดูในเดโม
1. กด "หมุนดูสิ" — วงล้อหมุนจริงมีแรงหน่วง
2. ดูการ์ดผล — มีบรรทัด "ทำไมถึงดี" น้ำเสียงอุ่นๆ
3. สลับหมวด ใกล้ๆตัว/ออกไปไกล/เจอผู้คน — เลือกตามใจที่ไหววันนั้น
4. กด "ทำแล้ว เก็บไว้ในสมุด" — ไปแปะในสมุดวันสบายใจ (loop ที่เคยขาด)
5. กด EN/TH — สลับทั้งหน้ารวมตัวหนังสือในวงล้อ
6. ลองบนมือถือ — ปุ่มใหญ่ วงล้อปรับขนาด

## 4) ถ้าชอบทิศนี้ ขั้นต่อไปกับของจริง
ย้ายของในเดโมเข้า `travel.html` แบบ minimal: เปลี่ยน `DEFAULT_ACTIVITIES` ใน `travel.js` เป็นชุดมีหมวด+เหตุผล, เพิ่มแถบหมวด+การ์ดผล+สมุดสะสม (ผูก localStorage คีย์ `healjai_*`), ใช้ปุ่ม share/bookmark เดิมจาก `common.js` (ไม่สร้างซ้ำ), พักบล็อกเกมช้อปไว้ (คอมเมนต์ ไม่ลบ) — ไม่แตะ common.js/โครงไฟล์ ไม่ push เอง รอเธอเทสมือถือก่อนค่อยลง TEAM-UPDATES
